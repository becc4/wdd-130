TicTacToe Game

June 2022

tictactoe.py

This TicTacToe Game was written as a warmup in CSE210 - Programming with Classes, and while I did not continue in that class in Spring 2022 semester, it started setting up some principles of classes such as the proper organization skills and the kind of writing we would be doing. For example, the print_board function is similar to the create_display classes we created that would direct the graphics to the games we made. This practice made me start to think about creating plans before programming with python, since there were more connected and moving elements in this program than in my previous ones.
